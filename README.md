# Excel-Graphing-Macro
Graphing Macros for Excel: Bar graphs with individual values and SEMs calculated and added; Repeated measures line graphs with SEMs calculated and added from long or wide data; converstion of long to wide data or wide to long.

Uploaded Excel Macros (.bas files) and Read-Mes
Macro's for generating individual value bar charts and line charts for repeated measures (made on Office365: compatibility with other versions is unknown). Macro's add SEMs. Macro for converting long data to wide data or visa versa. Read Me files and macro's were generated by vibe instructions with Co-Pilot. They seem to work well.

Instructions:  The macros present instructions on the format of the data and display variables (columns) separated by a "|". For example: Group 1 | Group 2 | Outcomes. Your grouping variables must be immediately to the left of your outcomes in wide-format. Repeated measures graphing expects animal IDs: Sample ID | Group | Outcomes. 

For the Bar Charts, you have the option of splitting data into multiple graphs or combining into one graph for multi-variable stuff. You can select a one-way or two-way anova design. You can choose to cluster data based on outcomes or based on groups depending on visualization preferences.

Individual values are designed to align with the bar center and separate if overlapping values are detected. All data points for each series are automatically calculated and organized and can be manually changed after manually setting the data range of interest, if desired. The individual values are added to the right Y axis while the bars are added to the left Y axis. When changing the chart range, or if you need to for whatever reason, ensure both axis have the same range and values for accurate reporting. The macro should first set the right Y axis range based on the range of the individual values, then match that range on the left Y axis to sensure appropriately aligned bars.

SEM values are calculated from the data and applied to the graph by default. Other error bar types will need to be manually applied if desired. 

A long-to-wide converter was added for convenience. 

To install macro's (files that end in .bas) that will be available between spreadsheets. 

Select: view -> Macros -> Record Macro -> Store macro in: Personal Macro Workbook -> OK

Then immediately turn off macro recording: Macros -> Turn off macro recording. 

*this will generate a folder where a macro excel file can be created, sored, and manipulated. if interested in manipulating these data can usually be found by pressing Windows + R, then pasting %AppData%\Microsoft\Excel\XLSTART . The PERSONAL.XLSB file stores the imported macros and can be transfered between devices. Multiple files can be added and stored here if desired. If an error occurs "Out of Memory" it may be useful to generate a new XLSB file here.

To begin importaing Macros to make them a button. 

Press: Alt + F11 -> select VBAProject (PERSONAL.XLBS) -> File -> Import -> select macro of interest -> save file.

Return to Excel Workbook -> Right click or push the down arrow in the Quick Access Bar at the top -> More Commands -> change "Choose Commands From:" to Macros
