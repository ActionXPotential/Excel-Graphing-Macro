# Excel-Graphing-Macro
Graphing Macros for Excel: Bar graphs with individual values and SEMs calculated and added; Repeated measures line graphs with SEMs calculated and added from long or wide data; converstion of long to wide data or wide to long.

Uploaded Excel Macros and Read-Mes
Macro's for generating individual value bar charts and line charts for repeated measures. Macro's add SEMs. Macro for converting long data to wide data or visa versa. Read Me files and macro's were generated by vibe instructions with Co-Pilot. They seem to work well.

Instructions when operating the macros separate expected variables by a "|". For example: Group 1 | Group 2 | Outcomes. It is useful to ensure your grouping variables are immediately to the left of your outcomes in wide-format. Repeated measures graphing expects animal IDs: Sample ID | Group | Outcomes. 

For the Bar Charts, you have the option of splitting data into multiple graphs or combining into one graph for multi-variable stuff. You can select a one-way or two-way anova design. You can chose to cluster data based on outcomes or based on groups depending on visualization preferences. 

Individual values are designed to align with the bar center and separate if overlapping values are detected. All data points for each series are organized and can be manually changed after manually setting the data range of interest, if desired. The individual values are added to the right Y axis while the bars are added to the left Y axis. When changing the chart range, or if you need to for whatever reason, ensure both axis have the same range and values for accurate reporting. 

SEM values are calculated by the data and automatically entered in. SD bars or other will need to be manually changed if desired.

A long-to-wide converter was added for convenience. 

To install macro's that will be available between spreadsheets. 

Select: view -> Macros -> Record Macro -> Store macro in: Personal Macro Workbook

Then immediately turn off macro recording: Macros -> Turn off macro recording.

Press: Alt + F11 -> select VBAProject (PERSONAL.XLBS) -> File -> Import -> select macro of interest -> save file.

Return to Excel Workbook -> Right click or push the down arrow in the Quick Access Bar at the top -> More Commands -> change "Choose Commands From:" to Macros

Select the added Macro (The name may be different, CoPilot gave some funky names, but it should start with PERSONAL.XLSB).

Modify the macro name and icon as desired and a button will appear on top. Follow the prompts. Get graphs. Be Happy
